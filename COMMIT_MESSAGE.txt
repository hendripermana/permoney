## Core Runtime Upgrades
- Ruby: 3.4.4 → 3.4.7 (Oct 7, 2025)
  * PRISM parser enabled by default

- Bundler: 2.6.9 → 2.7.2 (Sep 9, 2025)

  * Enhanced dependency resolution

- RubyGems: 3.6.9 → 3.7.2 (Sep 9, 2025)
